# DOC-LINK - 스마트 의료 매칭 플랫폼 MVP

병·의원과 의사(대진의)를 스케줄 기반으로 연결하는 의료인 전용 근무 매칭 플랫폼입니다.

## 프로젝트 개요

본 프로젝트는 단기 또는 일자별 대진이 필요한 병원과 근무 가능한 의사를 연결하는 매칭 플랫폼입니다. 기존의 일반적인 게시판형 구인구직과 달리, '스케줄'을 핵심 축으로 하여 의사의 근무 가능 일정과 병원의 요청 일정을 매칭하고, 중복 근무를 원천 차단하는 기능을 제공합니다.

## 주요 기능

### 1. 회원 관리
- 병원/의사 분리 가입 및 로그인
- 권한 분리 시스템
- 면허 인증 시스템

### 2. 의사 기능
- 프로필 관리 (전문과목, 경력, 근무 가능 지역)
- 근무 가능 요일/시간 스케줄 설정
- 모집 중인 대진 목록 조회 및 신청
- 신청한 대진 관리 (신청함 / 확정됨 / 완료)
- 캘린더 형태의 스케줄 관리
- 중복 근무 자동 차단
- 의사 간 일정 변경 요청

### 3. 병원 기능
- 대진 스케줄 등록 및 관리
- 날짜/시간/진료과/근무 조건 등록
- 진행 상태 관리 (모집중 / 신청 의사 확인 / 확정 / 완료)
- 신청한 의사 중 한 명 선택하여 확정
- 신청 의사 목록 관리
- 전체 의사 일정 통합 캘린더 뷰
- 전체 그리드 뷰 (시간별 의사 일정 한눈에 보기)
- 의사 직접 배정 또는 제안하기 방식
- 미결 일정 알림 기능 (연두색 음영 + 시각적 효과)
- 의사 간 일정 변경 요청 승인

### 4. 매칭 시스템 (1:N 구조)
- 병원이 대진 등록 → 다수의 의사가 신청 → 병원이 한 명 선택하여 확정
- 병원이 의사를 직접 배정 가능
- 스케줄 기반 자동 매칭 (의사 조건과 일치하는 대진 추천)
- 조건 일치 시 알림 제공
- 중복 근무 자동 차단

## 배포

### Vercel 배포

이 프로젝트는 Vercel을 통해 정적 사이트로 배포됩니다.

**Vercel 배포 설정:**
- Framework Preset: `Other` 또는 `Static Site`
- Root Directory: `./`
- Build Command: (비워두기)
- Output Directory: `./`
- Install Command: `npm install` (선택사항)

### 로컬 개발

```bash
# 의존성 설치
npm install

# 로컬 서버 실행
npm run serve
# 또는
npx http-server -p 8080 -c-1
```

서버는 http://localhost:8080 에서 실행됩니다.

## 주요 기능 상세

### 의사 페이지
1. **doctor-dashboard.html** - 의사 대시보드
   - 모집 중인 대진 목록 조회 및 신청
   - 내 신청 목록 (신청함/확정됨/완료)
   - 통계 정보
   - 캘린더 뷰 (날짜 클릭 시 시간별 타임라인으로 이동)
   - 일정 등록 기능

2. **doctor-profile.html** - 프로필 관리
   - 기본 정보 수정
   - 전문과목, 경력 설정
   - 근무 가능 지역 관리
   - 근무 가능 시간 설정 (30분 단위)

3. **schedule-timeline.html** - 시간별 타임라인
   - 30분 단위로 일정 확인
   - 시간대별 일정 추가
   - 날짜별 탐색

4. **shift-apply.html** - 대진 신청
   - 병원 정보 확인
   - 대진 상세 조건 확인
   - 신청하기 기능

### 병원 페이지
1. **hospital-dashboard.html** - 병원 대시보드
   - 통합 캘린더: 모든 의사의 일정을 한눈에 확인
   - 전체 그리드: 시간별 의사 일정을 테이블 형태로 확인
   - 대진 관리: 모집중/신청 의사/확정됨/완료 탭
   - 변경 요청: 의사 간 일정 변경 요청 승인
   - 미결 일정 알림: 3일 이내 미결 일정 연두색 음영 + 시각적 효과
   - 통계 정보
   - 의사별 필터링

2. **hospital-timeline.html** - 시간별 타임라인
   - 전체 의사의 시간별 일정 확인
   - 30분 단위 세분화 표시
   - 중복 부킹 경고 표시

3. **shift-register.html** - 대진 등록
   - 제안하기: 의사가 자원하도록 공개
   - 직접 배정: 의사를 직접 선택하여 배정
   - 중복 부킹 자동 방지

4. **shift-detail.html** - 대진 상세
   - 신청한 의사 목록 확인
   - 의사 선택 및 확정
   - 중복 부킹 체크
