# DOC-LINK - 스마트 의료 매칭 플랫폼 MVP

병·의원과 의사(대진의)를 스케줄 기반으로 연결하는 의료인 전용 근무 매칭 플랫폼입니다.

## 프로젝트 개요

본 프로젝트는 단기 또는 일자별 대진이 필요한 병원과 근무 가능한 의사를 연결하는 매칭 플랫폼입니다. 기존의 일반적인 게시판형 구인구직과 달리, '스케줄'을 핵심 축으로 하여 의사의 근무 가능 일정과 병원의 요청 일정을 매칭하고, 중복 근무를 원천 차단하는 기능을 제공합니다.

## 주요 기능

### 1. 회원 관리
- 병원/의사 분리 가입 및 로그인
- 권한 분리 시스템
- 면허 인증 시스템

### 2. 의사 기능
- 프로필 관리 (전문과목, 경력, 근무 가능 지역)
- 근무 가능 요일/시간 스케줄 설정
- 모집 중인 대진 목록 조회 및 신청
- 신청한 대진 관리 (신청함 / 확정됨 / 완료)
- 캘린더 형태의 스케줄 관리
- 중복 근무 자동 차단

### 3. 병원 기능
- 대진 스케줄 등록 및 관리
- 날짜/시간/진료과/근무 조건 등록
- 진행 상태 관리 (모집중 / 신청 의사 확인 / 확정 / 완료)
- 신청한 의사 중 한 명 선택하여 확정
- 신청 의사 목록 관리

### 4. 매칭 시스템 (1:N 구조)
- 병원이 대진 등록 → 다수의 의사가 신청 → 병원이 한 명 선택하여 확정
- 스케줄 기반 자동 매칭 (의사 조건과 일치하는 대진 추천)
- 조건 일치 시 알림 제공
- 중복 근무 자동 차단

## 페이지 설명

### 공통 페이지
1. **index.html** - 랜딩 페이지
   - 서비스 소개
   - 주요 기능 설명
   - 로그인/회원가입 링크

2. **login.html** - 로그인
   - 병원/의사 선택
   - 이메일/비밀번호 로그인

3. **signup.html** - 회원가입
   - 3단계 가입 프로세스
   - 병원/의사 구분
   - 약관 동의

### 의사 페이지
4. **doctor-dashboard.html** - 의사 대시보드
   - 모집 중인 대진 목록 조회 및 신청
   - 내 신청 목록 (신청함/확정됨/완료)
   - 통계 정보
   - 캘린더 뷰

5. **doctor-profile.html** - 프로필 관리
   - 기본 정보 수정
   - 전문과목, 경력 설정
   - 근무 가능 지역 관리
   - 근무 가능 시간 설정

6. **schedule.html** - 스케줄 관리
   - 캘린더 형태 일정 확인
   - 일정 추가/수정

7. **shift-apply.html** - 대진 신청
   - 병원 정보 확인
   - 대진 상세 조건 확인
   - 신청하기 기능

### 병원 페이지
8. **hospital-dashboard.html** - 병원 대시보드
   - 대진 관리 (모집중/신청 의사/확정됨/완료)
   - 신청한 의사 목록 확인 및 확정
   - 통계 정보
   - 캘린더 뷰

9. **hospital-profile.html** - 병원 프로필 관리
   - 병원 정보 수정
   - 진료 과목 설정
   - 담당자 정보

10. **shift-register.html** - 대진 등록
    - 근무 일정 등록
    - 진료과, 시간 설정
    - 필수 자격 요건 설정

11. **shift-detail.html** - 대진 상세
    - 신청한 의사 목록 확인
    - 의사 확정/거절

## 사용 방법

1. 브라우저에서 `index.html` 열기 또는 개발 서버 실행
   ```bash
   npm run serve
   ```
2. "시작하기" 버튼 클릭하여 로그인
3. 테스트 계정으로 로그인:
   - **의사 계정**: doctor@example.com / doctor123
   - **병원 계정**: hospital@example.com / hospital123
4. 사용자 유형 선택 (의사/병원)
5. 각 대시보드에서 기능 이용

## 테스트 계정

### 의사 계정
- 이메일: `doctor@example.com`
- 비밀번호: `doctor123`

### 병원 계정
- 이메일: `hospital@example.com`
- 비밀번호: `hospital123`

> 로그인 페이지 하단에 테스트 계정 정보가 표시됩니다.
